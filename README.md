Database (SSMS)
Ik heb eerst een database aangemaakt met tabellen gebaseerd op mijn ERD. De tabellen zijn Film, Serie, Genre, Gebruiker, Film_Genre en Serie_Genre. De koppeltabellen Film_Genre en Serie_Genre zijn er omdat een film meerdere genres kan hebben en een genre bij meerdere films kan horen. Daarna heb ik testdata ingevoegd zoals films (The Dark Knight, Pulp Fiction etc.) en series (Breaking Bad, Friends etc.) en die gekoppeld aan genres zoals Thriller, Comedy en Fantasie.
